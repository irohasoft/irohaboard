# iroha Board

iroha Board は日本で生まれたオープンソースのeラーニングシステム（LMS）です。
シンプルでフラットな構造と、使いやすいユーザインターフェイスが特徴で、手軽に独自のeラーニングシステムが構築できます。

## 公式サイト
https://irohaboard.irohasoft.jp/

## 動作環境
* PHP : 5.4以上
* MySQL : 5.1以上
* CakePHP : 2.10
